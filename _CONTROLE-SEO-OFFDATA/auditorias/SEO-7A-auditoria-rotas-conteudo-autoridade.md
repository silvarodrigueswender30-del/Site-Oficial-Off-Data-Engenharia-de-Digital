## Observações de linkagem atual

- A Home recebe forte papel de hub por Header, Produtos, Estratégias, Newsroom e CTA final.
- A seção Produtos já concentra a descoberta dos serviços, mas pode reforçar links textuais contextuais entre serviços.
- A seção Estratégias agora tem seis cards e funciona como camada de autoridade/nicho.
- O Footer pode ganhar, em fase futura, links para serviços comerciais principais e consultoria gratuita.
- A página futura `/consultoria-gratuita` deve ser planejada como rota de conversão e ponto de destino para CTAs editoriais.

## Oportunidades prioritárias

1. Criar página `/consultoria-gratuita` como captura de alta intenção.
2. Criar satélites locais para "site para clínica", "site para imobiliária", "como aparecer no Google em Uberlândia" e "site não gera clientes".
3. Reforçar linkagem das estratégias para serviços correspondentes.
4. Criar rotina GMB e backlinks locais para ampliar sinais de autoridade fora do site.
5. Usar Search Console para acompanhar impressões por consulta e priorizar ajustes de title/meta por CTR.
